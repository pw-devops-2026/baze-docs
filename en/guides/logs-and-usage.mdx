After you send model requests, use **Logs** and **Dashboard** to inspect results, analyze failures, and control cost.

## View request logs

Open [**Logs**](https://www.powertokens.ai/en/logs) to inspect API requests.

![PowerTokens Logs page](/images/guides/powertokens-quickstart/en/logs.jpg)

Logs help you:

- View request history
- Track generated results
- Find request failures
- Analyze errors
- Debug request parameters

When you troubleshoot a request, first check the request time, model name, request parameters, and error details.

## View usage stats

Open [**Dashboard**](https://www.powertokens.ai/en/dashboard) to review resource usage.

![PowerTokens Dashboard usage stats](/images/guides/powertokens-quickstart/en/dashboard.jpg)

You can analyze usage by model and API key:

- Credits usage
- Request count
- Success rate
- Key-level usage

Use these metrics to control cost and understand production usage.
