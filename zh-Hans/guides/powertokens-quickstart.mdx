PowerTokens 为生成式 AI 应用提供统一接口。你可以通过同一个平台接入聊天、图像、视频和音频模型，并在控制台管理 Credits、API Key、调用日志和用量统计。

这篇指南帮助你完成第一次接入。建议按顺序完成下面三个任务。

## 1. 注册账号并充值 Credits

访问 [PowerTokens 官网](https://www.powertokens.ai/) 并创建账号。

你可以使用：

- Google 登录
- GitHub 登录
- 邮箱注册

注册完成后，进入 **Dashboard** 控制台。

打开 [**Billing**](https://www.powertokens.ai/zh-Hans/billing) 页面，选择适合你的购买套餐。

![PowerTokens Billing 页面](/images/guides/powertokens-quickstart/billing.png)

Credits 是你的账户余额。调用模型时，平台会根据模型用量从 Credits 中扣费。

当前支付方式包括：

- 信用卡
- PayPal

## 2. 创建 API Key 并调用模型

充值完成后，创建 API Key，并通过平台 API 或 **Playground** 发起第一次请求。

你会完成：

- 创建并保存 API Key
- 设置 Key 的额度、重置周期和过期时间
- 在 **Models** 中选择模型
- 使用 `https://api.powertokens.ai` 调用平台接口
- 在 **Playground** 中调试 Prompt 和参数

继续阅读：[创建 API Key 并调用模型](/zh-Hans/guides/api-key-and-model-call)。

## 3. 查看日志和用量统计

发起请求后，使用 **Logs** 和 **Dashboard** 追踪调用结果、排查错误并控制成本。

你可以查看：

- 请求记录
- 生成结果
- 错误信息
- Credits 消耗
- 请求次数和成功率
- API Key 维度的用量

继续阅读：[查看日志和用量统计](/zh-Hans/guides/logs-and-usage)。
