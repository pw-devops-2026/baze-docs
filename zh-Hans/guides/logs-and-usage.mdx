title: "查看日志和用量统计"
description: "使用 Logs 和 Dashboard 追踪请求、排查错误并控制 Credits 消耗。"
---

发起模型请求后，你可以在 **Logs** 和 **Dashboard** 中查看调用结果、分析异常并控制成本。

## 查看调用日志

打开 [**Logs**](https://www.powertokens.ai/zh-Hans/logs) 页面，查看 API 请求记录。

![PowerTokens Logs 页面](/images/guides/powertokens-quickstart/logs.png)

日志中心可以帮助你：

- 查看请求记录
- 追踪生成结果
- 定位调用异常
- 分析错误信息
- 调试请求参数

排查问题时，先确认请求时间、模型名称、请求参数和错误信息是否符合预期。

## 查看用量统计

打开 [**Dashboard**](https://www.powertokens.ai/zh-Hans/dashboard) 页面，查看资源消耗情况。

![PowerTokens Dashboard 用量统计](/images/guides/powertokens-quickstart/dashboard.png)

你可以从模型和 API Key 维度分析用量：

- Credits 消耗
- 请求次数
- 成功率
- Key 消耗情况

这些数据可以帮助你控制成本并分析业务使用情况。
